# 🚗 Road Accident Analysis Dashboard (Excel)

This project presents an interactive Road Accident Dashboard built using Microsoft Excel.

## 📊 Key Features
- Dynamic filtering using slicers
- State-wise analysis of accident data
- KPI cards with real-time updates
- Donut charts and trend analysis
- Clean and professional dashboard design

## 🔍 Insights
- Slight injuries account for the majority of casualties
- Cars contribute the highest number of accidents
- Daytime accidents are more frequent than nighttime
- Road type and surface play a key role in accident patterns

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- Data Visualization
